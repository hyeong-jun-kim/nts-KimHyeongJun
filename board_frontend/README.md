# board_frontend

## Project setup
1. npm 설치
```
npm install
```

2. vue-router 설치
```
npm i --save vue-router
```

3. bootstrap vue 설치
```
npm install vue bootstrap bootstrap-vue
```

4. axios 설치
```
npm install --save axios
```

5. sass 설치
```
- npm i sass-loader
- npm i node-sass
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

npm run lint
